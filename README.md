## How To Run ...🏃...
- Download and Unzip The File
- Run the below code in both the directory (Frontend and Backend)
- Run both servers (Frontend and Backend) 


## Code For Installing Packages 📦
```js
      #Install Me
      cd frontend 
      npm i 
      
      cd backend
      npm i       
```
## Code For Running Server 📦
```js
      #Install Me
      cd frontend 
      npm start 
      
      cd backend
      npm start      
```
